# claude-tools — repo notes

When you need codebase context, check `openwiki/` first before searching the tree — start at `openwiki/index.md`.

This repo is a Claude Code plugin marketplace (`.claude-plugin/marketplace.json`), not an application: there is no `build/verify.ps1` here, and the loop-harness gate does not apply to changes in this repo itself. Match the existing style of whatever plugin's files you're editing; keep prose in UK English with no em-dashes, per the conventions the plugins themselves document.
